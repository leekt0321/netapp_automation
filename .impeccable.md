# Impeccable 디자인 컨텍스트

## 제품

- 이름: `netapp_automation`
- 유형: NetApp 스토리지 로그 업로드, 파싱, 검토, 사이트/상태 관리를 위한 내부 운영 콘솔

## 대상 사용자

- 주요 사용자는 Baobab Technology에서 일하는 NetApp 스토리지 엔지니어다.
- 이들은 업로드된 로그를 검토하고, 장비 구성을 확인하고, 운영 상태를 추적하는 작업을 오랜 시간 반복해서 수행한다.
- 장식적인 UI보다 속도, 명확성, 안정성, 낮은 탐색 피로도를 더 중요하게 여긴다.

## 핵심 사용 시나리오

- 수집한 NetApp 로그를 시스템에 업로드한다.
- 업로드된 로그에서 의미 있는 장비 정보를 추출하고 표시한다.
- 구조화된 목록과 상세 화면을 통해 장비 구성과 현재 상태를 검토한다.
- 사이트 단위와 스토리지 단위의 운영 정보를 한 곳에서 관리한다.
- 로그가 어떤 장비에 속하는지, 무엇이 바뀌었는지, 어떤 점을 주의해야 하는지를 빠르게 파악한다.

## 브랜드 성격

- 차분함
- 정확함
- 운영 중심
- 신뢰감
- 저피로
- 단순함

이 인터페이스는 화려한 분석 대시보드가 아니라 믿고 사용할 수 있는 엔지니어링 운영 콘솔처럼 느껴져야 한다.

## UX 우선순위

- 로그 기반의 밀도 높은 정보를 빠르게 훑어볼 수 있게 만든다.
- 장시간 사용 시 눈의 피로를 줄인다.
- 목록, 요약, 상세 보기 사이를 빠르게 오갈 수 있게 지원한다.
- 장식보다 정보 위계를 더 분명하게 드러낸다.
- 중요한 장비 메타데이터, 상태, 구조 정보를 과하지 않게 항상 잘 보이도록 유지한다.

## 비주얼 방향

- 전체 분위기: 차분하고 집중할 수 있으며 가독성이 높아야 한다.
- 테마 방향: 장시간 업무에 적합한 밝은 톤의 중립적 인터페이스
- 레이아웃 스타일: 구조적인 분할 화면과 데이터 중심 구성, 한눈에 보기 좋은 구성
- 감정적 목표: 사용자가 몇 초 안에 현재 위치를 이해하고 통제감을 느껴야 한다.

## 디자인 원칙

- 파싱된 데이터를 즉시 이해할 수 있도록 대비와 그룹화를 우선한다.
- 색상은 심각도, 상태, 선택 등 의미가 있을 때만 절제해서 사용한다.
- 번들거리는 대시보드 스타일, 과한 그라데이션, 장식용 카드 남용은 피한다.
- 비교 작업에 유리하도록 명확한 구획, 고정된 맥락 정보, 안정적인 정렬을 선호한다.
- 테이블, 리스트, 배지, 요약 영역은 차분하고 일관된 느낌을 유지해야 한다.

## 반드시 기억에 남아야 하는 점

- 이 제품은 복잡한 스토리지 로그 정보를 유난히 읽기 쉽고 관리하기 쉬운 도구로 느껴져야 한다.
- 이 제품의 핵심 인상은 "운영의 차분함"이다. 즉, 밀도 높은 기술 데이터를 명확하고 피로감 낮게 보여주는 것이 가장 중요하다.
